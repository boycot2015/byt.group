# 基于Astro开发的个人兴趣爱好合集网站

## 技术栈

Astro、Svelte、Vue3、React、twindcss

## 🚀 目录结构

```text
/
├── public/
│   └── favicon.svg
├── src/
│   ├── components/
│   │   └── Card.astro
│   ├── layouts/
│   │   └── Layout.astro
│   └── pages/
│       └── index.astro
└── package.json
```
## 启动部署

```bash
pnpm install // 安装,node>=18.x
npm run dev // 启动
npm run build // 打包生产
npm run preview // 预览
```
